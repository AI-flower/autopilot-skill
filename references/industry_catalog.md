# AI Praxis - Industry & Scenario Catalog

This catalog covers common scenarios across industries. Used by the skill to recommend
actions to users who don't know what to do, and to classify incoming requests.

---

## 1. Office Automation / 办公效率

### Email / 邮件
- Send daily/weekly team reports via email
- Auto-reply to emails based on rules
- Batch send personalized emails (mail merge)
- Set up email forwarding and filtering

### Documents / 文档
- Convert documents between formats (PDF, DOCX, MD, HTML)
- Batch rename files with patterns
- Summarize long documents/reports
- Generate meeting minutes from notes
- Create presentation slides from outline

### Calendar & Tasks / 日程与任务
- Set up recurring reminders
- Create task lists from meeting notes
- Sync tasks between platforms (Todoist, Things, Reminders)

### Data Processing / 数据处理
- Clean and format CSV/Excel data
- Merge multiple spreadsheets
- Generate charts and reports from data
- Extract data from PDFs or images (OCR)

## 9. Design / 设计

### Product Prototype / 产品原型 ⭐
**触发关键词**: 原型图、产品设计、界面设计、UI设计、交互设计、做个页面、prototype、mockup、wireframe
**执行模板**: 参见 `references/product_prototype_template.md`

标准输出：
- 可点击交互式 HTML 原型（所有页面均可跳转）
- 视觉完整的网页版（HTML + CSS，可直接在浏览器查看）
- 自动在浏览器中打开结果

常见场景：
- 为某个产品做完整原型（首页 + 核心功能页 + 详情页）
- 为已有需求文档生成可点击 demo
- 快速出一个 landing page / 落地页
- App 界面设计转网页版
- 做一个管理后台的界面原型

### UI/UX / 界面设计
- Generate color palettes and design tokens
- Design system setup (components, spacing, typography)
- Accessibility audit
- Dark mode adaptation

### Graphics / 图形设计
- Create social media graphics
- Design simple logos
- Generate icons and illustrations
- Create infographics from data
